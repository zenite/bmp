说明和术语表

Instruction and Glossary

  说明 Instruction
1)
关于表格 About the form
本表用于新供应商申请供应商代码。
The form is used for the code application request of a new supplier.
本表格必须完整填写（包括下拉列表选项）。
This form must be filled in completely (including the list options).
申请人填写表单提出申请，经总部采购供应部评价后，由管理层批复。
The applicant submits the request. The top management will approved the request with the comments of HQ Purchasing and Supply Department.
申请集团供应商必须经集团副总批准。
The request for the group level supplier should to be approved by the group EDGM.
2)
关于新供应商信息部分 About New Supplier Information
供应商名称为工商注册名称全称。
The supplier name should be the Registered Name.
没有英文名称，可以用中文全称拼音代替。
If no English name, Chinese pinyin can be replaced.
地址为工商注册地址
The address should be the Registered address.
3)
新供应商评价结果参见供应商评估程序要求
The fill in request of the assessment result need follow the supplier assessment procedure.
4)
若现有的资源满足或部分满足需求，则优先采用现有资源。
The existing resources will be priority considered, if they can meet or partially meet the demand.
5)
关于其它信息部分 About other information
一次性/非一次性采购 One-time/Multi-time
1- 非生产性且一次性采购供应商 Non-production Supplier of One-time purchase
9~2- 非生产性且非一次性采购供应商（从9到2依次使用）Non-production Supplier of Multi-time purchase,from 9 to 2
A~W- 生产性且非一次性采购供应商（制造地/发货地/代理），从A到W依次使用 Production supplier of multi-time purchase(including manufacture side/shipping side/agency),from A to W
X- 生产性且一次性采购供应商(包括代理）Production Supplier of One-time purchase(Including agency)
